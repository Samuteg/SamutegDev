Este blog vem com a integração [@astrojs/mdx](https://docs.astro.build/pt-br/guides/integrations-guide/mdx/) já configurada no `astro.config.mjs`. Se preferir não usar MDX, basta remover a integração do arquivo de configuração.

## Por que MDX?

MDX é uma variação do Markdown que suporta JavaScript e JSX embutidos. Isso permite [misturar componentes de UI ao conteúdo](https://docs.astro.build/pt-br/guides/integrations-guide/mdx/#mdx-no-astro) — útil para alertas, botões interativos ou widgets personalizados.

Se você já tem conteúdo em MDX, migrar para Astro tende a ser direto.

## Exemplo

Veja como importar e usar um componente Astro dentro de um arquivo MDX. Ao abrir esta página no navegador, o botão abaixo deve ser clicável:

import HeaderLink from '../../components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('clicou!')">
	Componente embutido em MDX
</HeaderLink>
